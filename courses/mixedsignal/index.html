Mixed-Signal Test and Product Engineering Course

The Mixed-Signal Test and Product Engineering course combines analog and digital testing techniques. Particular attention is placed on integrating the analog source/measurement units with the digital source/measurement units to obtain the correct timing for driving and measuring the mixed-signal parts. A significant amount of time is required to teach the typical circuit structures underlying the data converters since students do not have this background and the circuit changes the testing techniques. Like the Analog TPE course, students are expected to use statistical analysis to optimize test programs for speed versus accuracy.

List of Course Topics

Weeks Segment Topics Covered
3 Overview of Test and Product Eng. Using a comparator Why test?
Datasheets as specification list
Common test equipment
Common comparator DC tests
Data analysis, including repeatability, reproducibility, GRR, and guardbanding
4 DAC Testing Overview of different DAC structures
Voltage references
Resistor trimming
Datasheets and common specifications for DACs
Common DC tests (DC gain/gain error, offset voltage, LSB step size)
Transfer function tests (DNL, INL, monotonicity)
Dynamic tests (conversion time, rise/fall time, signal-to-noise ratio, glitch energy, clock feedthrough)
3 ADC Testing Overview of different ADC structures
Datasheets and common specifications for ADCs
Common DC tests (DC gain, offset voltage)
Transfer curve tests (DNL and INL using histogram based testing)
Dynamic considerations (conversion time, recovery time, sampling frequency)

List of Labs

Lab Topic Instructional Goals
1 Program Order and ATE Resources Students become familiar with ATE equipment, test board, and programming environment
2 Continuity Students create a function, use analog resources, datalog, run the program on ATE, and critically think about results
3 Comparator Vos Students gain more practice in skills from lab 2
4 Slew Rate Test Students learn the clocking set-up, the AWG function, and the time-measurement unit
5 DAC Continuity Students learn to set-up the digital resources
6 DAC Gain Error and Vos Students use the vector editor to simultaneously control the digital input and the analog measurement to measure Gain Error and Vos
7 DAC INL and DNL Students use the vector editor to test all digital codes and calculate DNL and INL. Students perform a repeatability study with this data.
8 to 10 Project: ADC Transfer Function Test Students write the test function to create the DC transfer curve of the ADC using a histogram test method. Emphasis is on trying multiple methods and learning from the methods rather than a complete, correct solution.
