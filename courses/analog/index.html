Analog Test and Product Engineering Course

The Analog Test and Product Engineering course teaches the foundations of test and product engineering using an op-amp. Student familiarity with the op-amp allows more time to be focused on the test and product engineering concepts and less time on circuit foundations. Yet testing a simple op-amp requires feedback control, a thorough understanding of the data sheets, a variety of part specifications and instrument requirements, and provides the opportunity to use digital-signal processing testing techniques.


List of Course Topics

Weeks Segment Topics Covered
1 Overview of Test and Product Engineering Why test?
Fabrication process
Throughput, capacity, test time, and impacts on part cost
1 ATE Test Equipment Typical equipment (source/measurement units, Arbitrary Waveform Generators, Digitizers, Relays) Schematics, limitations, and time-accuracy trade-offs
3 DC Tests Common tests (Continuity, supply currents, offset-voltage, gain, PSRR, CMRR)
Test board generation, test code generation
Trade-offs of different testing methodologies
Economic trade-offs
3 AC Tests Common tests (gain, frequency response, phase response, harmonics, noise)
Sampling theory for multi-tone coherent signal testing
1 Measurement Accuracy Repeatability, reproducibility, correlation, calibration
1 Data Analysis Statistical analysis of data, CP, CPK, GRR, guardbanding for error

List of Labs

Lab Topic Instructional Goals
1 Program Order and ATE Resources Students become familiar with ATE equipment, test board, and programming environment
2 Continuity Students create a function, use analog resources, datalog, run the program on ATE, and critically think about results
3 Power Supply Current Students gain more practice in skills from lab 2
4 Input Offset Voltage Students use a false-summing amp for feedback control and experience the challenges in obtaining high-precision measurements
5 Open Loop Gain Students use a nulling-amp for feedback control and gain more experience in obtaining high-precision measurements
6 Pattern-Based Testing Students control instruments using a test pattern (using clocks) and measure Vos, CMRR, and PSRR in one test
7 AC Gain-Bandwidth Students perform a coherent multi-tone test to measure gain-bandwidth specification
8 to 10 Project: Design test program for new chip Students receive a datasheet for a new part and generate: 1) a test plan, 2) production-level test code, and 3) data analysis (GRR) indicating that the program is ready for production
