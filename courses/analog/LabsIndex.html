  • Analog Device Interface Board (DIB)

    • The analog DIB is used to test a number of general purpose op-amps in an 8-pin DIP package. The DIB contains circuitry to test many of the specifications using a false-summing junction or a nulling amplifier. This gives students more experiences in test methodology. The DIB also allows the op-amp to be tested in the noninverting negative feedback configuration so that AC tests can be performed.


    Page 1 shows the circuitry used to test site 1. The red numbers in boxes are the relay control-bit numbers.
    Page 2 shows the circuitry used to test site 2. The same relay control bits effect relay position on both sites.
